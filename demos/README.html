yFiles.NET Programming Samples Overview

The yFiles.NET programming samples are extensive Visual Studio projects with full C# source code that present the functionality of the yFiles.NET diagramming library. All programming samples can be found in subdirectories of the current directory. Most samples are self-contained, i.e., they consist of a single Visual Studio project.

Running a Sample

To open a specific programming sample, the corresponding Visual Studio project file (usually named [demoname].csproj) can be found in the demo's directory, which also uses the [demoname] naming scheme.
Note that samples that demonstrate related aspects are grouped together in solution folders.

All demos can also be accessed by opening the solution file (yFiles Demos.sln) in this folder.

Programming Samples Categories

Category Folder Description
BPMN Demos BPMN Source code samples that demonstrate how to implement BPMN layout and editing features.
Compatibility Classes Compatibility Support project to facilitate migration from yFiles.NET 4.4 to yFiles.NET 5.0.
Complete Demos Complete Source code samples that demonstrate the interaction between the viewer component and the layout algorithms.
Data Binding Demos DataBinding Source code samples that demonstrate binding graph elements to business data.
InputMode Demos Input Source code samples that demonstrate how to configure specific aspects of the InputMode functionality.
Layout Demos Layout Source code samples demonstrate the basic usage of the yFiles.NET layout algorithms.
Style Demos Style Source code samples that demonstrate how to customize the visual representation of yFiles.NET graph elements.
Utility Projects for Demos Utils Utility projects for common functionality in demos.
Graph Viewer Demos View Source code samples demonstrate basic concepts related to the GraphControl component and the underlying graph model.
yEd.NET Graph Editor Demo yEd.NET Complex demo application that showcases most features that are present in yFiles.NET. yEd.NET will help you to interactively explore many capabilities of both the yFiles.NET Layout and the yFiles.NET Viewer functionality.
