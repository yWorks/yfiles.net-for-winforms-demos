Organization Chart

Mouse Interaction

Mouse Wheel
Changes the zoom level of the view.
Left Mouse Drag
Moves the diagram within the view.
Left Click on Employee
Selects the employee in both the Chart and the Structure View. Shows the corresponding data in the properties view.
Double Click
Zoom in on the selected employee.
Hover over Employee
Provides tool tip for the employee.
Hover over Buttons
Provides tool tip for actions.
Hover over Circle
Provides tool tip info for employee state.
Click on Buttons
Activates corresponding actions to change the visible structure.

Key Bindings

To use keyboard navigation the chart needs to have the focus.

Left Arrow Key
Selects a node to the left of the currently selected node
Right Arrow Key
Selects a node to the right of the currently selected node
Up Arrow Key
Selects a node above the currently selected node
Down Arrow Key
Selects a node below the currently selected node
CTRL + Left Arrow Key
Moves the diagram to the right
CTRL + Right Arrow Key
Moves the diagram to the left
CTRL + Up Arrow Key
Moves the diagram down
CTRL + Down Arrow Key
Moves the diagram up
+ Key
Zoom In
- Key
Zoom Out
Enter
Zoom in on the selected employee.
CTRL + PageUp Key
Shows the parent of the current employee.
CTRL + PageDown Key
Hides the parent of the current employee.
CTRL + Add Key
Shows the children of the current employee.
CTRL + Subtract Key
Hides the children of the current employee.

Overview

Mouse Interaction

Mouse Drag highlighted region
Move the diagram in the main view
Mouse Drag corner of highlighted region
Change zoom level of the main view

Structure

Mouse Interaction

Click on Employee
Selects the employee in both the Organization Chart and the Structure View. Shows the corresponding data in the properties view.
Double Click on Employee
Same as Click. Moreover the Organization Chart is zoomed in on the selected employee. If the employee is not currently in the diagram it is set as the new root.
